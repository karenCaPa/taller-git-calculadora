# Calculadora - Taller Git

Proyecto de ejemplo para practicar Git y GitHub en pareja.

## Funcionalidades

- Suma de dos números.
- Resta de dos números.
- Multiplicación de dos números.
- División de dos números con validación de división por cero.
- Validación de campos vacíos antes de calcular.

## Trabajo colaborativo

- Uso de ramas para nuevas funcionalidades.
- Pull requests revisados por el compañero.
- Commits realizados por ambos colaboradores.
- Merges y eliminación de ramas después de integrar los cambios.
